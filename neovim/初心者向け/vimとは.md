# Vimとは
## はじめに
Vimとは何か？Vimはテキストエディタです。ただし、一般的なテキストエディタ(サクラエディタや秀丸、メモ帳、VSCodeなど)とは大きく違い、GUIではなくCUIで動作する物になります。

## Vimを使う必要があるのか？
Vimではなくて、普段使っているテキストエディタでいいじゃないか？という意見もあるかと思います。
ただ、Linux系のサーバーに入って作業を行うような職業の場合は(私です)基本的なVimの操作ぐらいは使えた方が良いです。
なぜかというと、Linuxのサーバー自体には大抵の場合viかVimしか入ってないからです。
Vimの操作ができないと、サーバー上に入ってファイルを編集しようとするのも大変です。
実際にはサーバーエンジニアの中にはVimの操作なんてわからん！！使うときに調べる！！という人も多いですが、普段使いにVimを使っておけば調べる時間も必要なくなりますし、バリバリのVimmerはファイルの編集速度が速くてとてもかっこいいです！(重要)
なのでみんなでVimを使ってVimmerを目指しましょう！！


## 最低限覚えておくべき操作

### ファイルの開き方

シェルから実行します。ファイルが存在している場合はファイルを開きます。ファイルがない場合は名前をつけた状態で開きます。

```
vim test.txt
```

とりあえず空の状態で開きたい場合は以下のようにファイル名をしていしないで実行します。

```
vim
```

vim上から開くには次のように実行します。

```
:e test.txt
```

---

### ファイルの保存

vim上で以下のように実行します

```
:w
```

名前をつけて保存するには以下のように実行します

```
:w test.txt
```

### Vimを終了する

vimから抜けるには以下のようにします

```
:q
```

ファイルを編集中の場合はファイルを保存しないと抜けれないので、保存して抜けるには下記のようにします

```
:wq
```

保存せずに強制的に抜けるには以下のようにします。

```
:q!
```

---

### vimの移動

vim上で移動するにはh,j,k,lを使用します。

| キー | 動作 |
| --- | --- |
| h | 左に移動 |
| j | 下へ移動 |
| k | 上へ移動 |
| l | 右へ移動 |


### 文字の入力

iキーを入力してカーソル位置の左側からインサートモードに推移し入力する

aキーを入力してカーソル位置の右側からインサートモードに推移し入力する

### ノーマルモードに戻る

ESCキーでノーマルモードに戻ることができます。なんか思ってたのと違う動きをし始めたとかなったらとりあえずESCキーを押しましょう

## モードについて

インサートモードとかノーマルモードという言葉が出てきてわからん！！となった人もいるかと思います。Vimでは色々なモードがあり、モード毎にキーでの操作が変わります。

基本的な操作だけであればノーマルモード、インサートモード、コマンドモード、ビジュアルモードを覚えておくと良いです。

### ノーマルモード

Vimを起動したときの最初のモード

カーソルの移動などはこのモードで行います。

他のモードからはESCキーでノーマルモードに戻ってくることが可能です。

### インサートモード

文字を入力するためのモードです。

### コマンドモード

保存やVimの終了など:から始まるもの、/ではじまる検索などを行うモードです。

キーを入力すると画面の下のコマンドラインに表示されます。

### ビジュアルモード

ノーマルモードからvキーを入力することで入れます。

範囲指定をするときに使用します。

## 終わりに
私も使いこなせているか微妙ですが、Vimはとても便利なエディタなので使いこなせるように頑張ろう！！
